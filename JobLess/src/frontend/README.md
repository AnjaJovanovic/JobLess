# JobLess Frontend

React + Vite aplikacija za kandidate i kompanije.

## Dokumentacija

```bash
# samo frontend (JSDoc)
npm run docs:frontend

# backend + frontend
npm run docs:all

# lokalni server: http://localhost:8000
npm run docs:serve
```

Izlaz: `docs/frontend/index.html` (relativno od root-a JobLess projekta).
